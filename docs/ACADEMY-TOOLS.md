# Academy Tools Roadmap

READ MASTER 같은 프랜차이즈 학원에 필요한 도구를 현재 Apps Script 프로젝트 기준으로 묶었습니다. 이 문서는 이후 계속 개발할 제품 범위를 정리하는 참조 문서입니다.

## P0 지금 붙어 있는 도구

1. 학부모 리포트 자동 생성
2. Docs/PDF 자동 생성
3. 학부모 공개 링크
4. 이메일 발송
5. 카카오 알림톡 공급사 webhook payload
6. 설명회/상담 리드 intake
7. 가맹점 대시보드
8. 빠른 시작 튜토리얼

## P1 바로 이어서 만들 것

1. 리드 상태 관리 CRM
처음 문의, 연락 완료, 테스트 예약, 리포트 발송, 등록 완료까지 관리

2. 레벨테스트 결과 입력 시트
테스트 점수와 관찰 코멘트를 입력하면 리포트 초안 자동 생성

3. 리드 -> 학생 변환
설명회 예약자나 상담 예약자를 Students 시트로 옮기는 전환 버튼

4. 학부모 링크 열람 로그
링크가 열렸는지, 언제 열렸는지, 몇 번 열렸는지 기록

5. 본사 표준 템플릿 배포
본사에서 템플릿 Doc ID와 문구 구조를 관리하고 가맹점은 선택만 하도록 구성

6. 카카오 템플릿별 분기
신규 설명회 후속, 테스트 결과 안내, 월간 리포트 안내를 별도 템플릿으로 관리

## P2 운영 SaaS로 가기 위한 도구

1. 가맹점 상담 파이프라인 보드
Kanban 형태로 설명회 예약 -> 상담 -> 테스트 -> 등록을 한눈에 관리

2. 설명회 퍼널 대시보드
유입 수, 예약 수, 참석 수, 테스트 진행 수, 등록 전환 수 추적

3. 월간 학부모 리포트 배치
주기적으로 반별 리포트를 자동 생성하고 예약 발송

4. 출결/과제 데이터 연동
수업 운영 시트나 LMS에서 출결/과제를 끌어와 리포트에 반영

5. 상담 예약 자동화
예약 링크 발송, 미응답 리마인드, 예약 완료 후 알림 자동화

6. 본사-가맹점 멀티캠퍼스 대시보드
가맹점별 리드 전환율과 리포트 운영 수준을 본사에서 집계

## 제품 묶음 기준

### 1. 입학 전환 묶음

- 설명회 퍼널
- 상담 예약
- 리드 intake
- 레벨테스트
- 학부모 리포트
- 등록 상담

### 2. 재원생 운영 묶음

- 월간 리포트
- 과제/출결 반영
- 카카오 안내
- 상담 기록

### 3. 본사 운영 묶음

- 템플릿 통제
- 지점별 현황
- 전환율 비교
- 캠페인 분석

## 장기 방향

현재 Apps Script는 가장 빠른 운영 도구입니다. 하지만 아래 조건이 늘어나면 Next.js/NestJS 같은 별도 SaaS로 이관하는 게 맞습니다.

- 가맹점 수가 많아짐
- 권한 관리가 복잡해짐
- 본사 승인 프로세스가 필요해짐
- 카카오/CRM/결제 연동이 깊어짐
- 실시간 대시보드 요구가 커짐

즉, 이 프로젝트는 `현장 운영 MVP`로 시작하고, 이후 `프랜차이즈 운영 SaaS`로 올라가는 출발점으로 보는 것이 맞습니다.
