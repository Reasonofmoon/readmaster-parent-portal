# READ MASTER 확장 도구 아이디어

`readmaster-funnel`의 설명회 퍼널과 현재 학부모 리포트 GAS를 같이 운영한다는 전제로 정리했습니다.

## P0 바로 붙일 것

1. 설명회 예약 접수 시트와 학부모 리포트 시트 연결
2. 레벨테스트 결과 입력 후 자동 리포트 생성
3. 학부모 링크 발송 로그와 열람 로그 추적
4. 상담 예약 URL을 리포트와 카카오 버튼에 포함
5. 가맹점별 템플릿 배포와 브랜드명 잠금

## P1 운영 확장

1. 가맹점 상담 CRM
2. 설명회 참석 후 미등록 학부모 리마인드
3. 월간 재원생 리포트 자동 생성
4. 반별 출결/과제 데이터를 리포트에 자동 반영
5. 캠퍼스별 리포트 발송 현황 대시보드
6. 본사 표준 문구 라이브러리
7. 신규 등록 전환률 추적
8. 카카오 템플릿 버전 관리

## P2 플랫폼 확장

1. 본사-가맹점 멀티테넌트 운영 포털
2. 설명회 퍼널 A/B 테스트
3. 커리큘럼 페이지 개인화
4. 학부모 상담 예약 자동화
5. 본사 광고 유입부터 등록까지 퍼널 분석
6. 재원생 추천/형제 등록 리퍼럴 프로그램

## 권장 제품 묶음

- 설명회 퍼널
- 상담 예약
- 레벨테스트 기록
- 학부모 리포트
- 카카오/이메일 전달
- 등록 전환 CRM
